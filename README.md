# KosorinStein
My own wolfenstein 3d clone

## Raycaster supports
- textured walls
- doors
- entites
- alpha blending ( semi transparent entities are also supported - in demo is semi transparent barrel)
- weapons
- you can see through windows
- dynamic lightning
- every lightning source can have own rgb parts
